overview

This presentation will show how to build a desktop GUI with Tkinter, including how to:
  set up a layout
  show content: text, images, shapes
  add styling: colors, borders
  interact with user input:
    button, radio button, option menu, event binding, text entry
  add animation

using features of Python such as:
  classes
  packages/modules
  partials
  built-in functions
  *args
